# 📦 Supply Chain Analytics - Demand Forecasting & Anomaly Detection

## 📌 Project Overview

This project demonstrates an end-to-end Supply Chain Analytics solution using Oracle SQL, Python, and Power BI.

The objective is to analyze sales and inventory performance, forecast future demand using ARIMA, detect anomalies in sales data, and build an interactive Power BI dashboard for business decision-making.
